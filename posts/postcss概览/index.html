postcss概览

简单来说，PostCSS是一个能将CSS转换为AST的Node模块，基于它能够开发js插件，使用js插件能够对css文件进行一些操作，比如：lint, autoprefix, polyfill…

更加应该将其理解为CSS后处理器，相对于less, sass等预处理器。但是一些插件仍可以实现类似于预处理器的功能。

  1. 比较常用的一些插件

    • autoprefixer
    • postcss-preset-env
    • postcss-pxtorem
    • postcss-px-to-viewport
    • …

  2. 使用方法

    例如在webpack中使用，使用postcss-loader

    webpack.config.js
    module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'postcss-loader'],
      }
    ]
  }
}
    postcss.config.js
    module.exports = {
  plugins: [
    require('autoprefixer'),
    require('postcss-nested')
  ]
}
